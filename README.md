# Hospital Operations SQL Analysis

## Project Overview
This project analyzes hospital operational efficiency using PostgreSQL.
It focuses on doctor workload, department performance, length of stay,
and treatment costs.

## Database Schema
- patients
- doctors
- departments
- admissions
- treatments

## Key Analyses
- Doctor workload ranking
- Average length of stay by doctor and department
- Treatment cost analysis
- High-cost admissions
- Executive KPI summary

## SQL Skills Demonstrated
- Multi-table joins
- CTEs
- Window functions
- Date arithmetic
- Aggregations and KPIs

## How to Run
1. Execute `schema.sql`
2. Run `insert_data.sql`
3. Run queries in `/sql` folder in order
